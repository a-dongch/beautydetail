# 뷰티디테일 (BeautyDetail) - 홍보 페이지

뷰티샵을 위한 프리미엄 홍보페이지 제작 서비스 소개 사이트입니다.

## 🎯 프로젝트 개요

**뷰티디테일**은 뷰티샵 원장님들을 위한 맞춤형 홍보 페이지 제작 서비스를 소개하는 원페이지 웹사이트입니다. 모던하고 세련된 디자인으로 서비스의 가치를 효과적으로 전달합니다.

### 주요 목적
- 뷰티샵 홍보페이지 제작 서비스 소개
- 다양한 가격 플랜 및 서비스 내용 안내
- 고객 문의 및 상담 유도
- SNS 광고 랜딩페이지로 활용 가능

## ✨ 주요 기능

### 완료된 기능

#### 1. **히어로 섹션**
- 화려한 그라데이션 배경 애니메이션
- 실시간 카운터 애니메이션 (제작 완료, 만족도, 24시간 대응)
- CTA 버튼을 통한 플랜 및 템플릿 섹션으로 이동
- 스크롤 인디케이터로 사용자 유도

#### 2. **서비스 소개 (About)**
- 6가지 차별점 소개 (카드 형태)
- 마우스 호버 시 3D 틸트 효과
- 스크롤 애니메이션 (AOS 라이브러리)
- 아이콘과 그라데이션을 활용한 시각적 매력

#### 3. **활용 방안 (Use Cases)**
- 4가지 활용 사례 소개
- 번호 순서대로 가독성 높은 레이아웃
- 각 카드별 고유한 컬러 테마
- 호버 시 슬라이드 효과

#### 4. **가격 플랜 (Pricing)**
- 3가지 플랜: Standard / Premium / Maintenance
- 프리미엄 플랜에 "MOST POPULAR" 배지
- 플랜 선택 시 자동으로 문의 폼으로 이동
- 플랜 상세 비교 아코디언 테이블
- 체크마크로 포함/미포함 항목 명확히 표시

#### 5. **템플릿 갤러리**
- Swiper 라이브러리를 활용한 슬라이더
- 반응형 그리드 (모바일 1개, 태블릿 2개, 데스크톱 3개)
- 호버 시 오버레이 효과 및 미리보기 버튼
- 각 템플릿에 태그 표시 (고급, 클래식, 모던 등)

#### 6. **문의 폼 (Contact)**
- 인스타그램 DM 문의 정보
- 빠른 응답, 신속 제작, 지속 지원 아이콘 통계
- 실시간 폼 입력 및 검증
- 플랜 선택 시 자동으로 선택한 플랜 프리셋
- 제출 시 알림 메시지

#### 7. **FAQ 아코디언**
- 5가지 자주 묻는 질문
- 클릭 시 열림/닫힘 애니메이션
- 한 번에 하나의 항목만 열림

#### 8. **푸터 (Footer)**
- 4개 컬럼: 브랜드 소개 / 빠른 링크 / 고객 지원 / 연락처
- 소셜 미디어 링크 (Instagram, Facebook, YouTube, Blog)
- 저작권 및 법적 정보
- 다크 그라데이션 배경

#### 9. **인터랙티브 기능**
- 네비게이션 바 스크롤 시 배경 변경
- 모바일 햄버거 메뉴
- 부드러운 스크롤 애니메이션
- 맨 위로 가기 버튼 (스크롤 500px 이후 표시)
- 전역 알림 시스템
- 카운터 애니메이션
- 3D 카드 틸트 효과
- 패럴랙스 효과

#### 10. **접근성 & UX**
- 키보드 네비게이션 지원
- ARIA 라벨 및 역할 속성
- 스크린 리더 호환
- 반응형 디자인 (모바일/태블릿/데스크톱)
- 로딩 최적화

## 🎨 디자인 특징

### 컬러 팔레트
- **Primary**: `#d4558d` (핑크)
- **Secondary**: `#7c5ccc` (퍼플)
- **Gradient**: 핑크에서 퍼플로 이어지는 그라데이션
- **Text**: `#1a1a2e` (다크 네이비)
- **Background**: 화이트 & 라이트 그레이

### 타이포그래피
- **Main Font**: Noto Sans KR (본문)
- **Display Font**: Playfair Display (제목)

### 애니메이션 라이브러리
- **AOS (Animate On Scroll)**: 스크롤 시 요소 등장 애니메이션
- **Swiper**: 터치 슬라이더 (템플릿 갤러리)
- **Custom Animations**: CSS 키프레임 & JS 애니메이션

## 📱 반응형 디자인

- **Desktop**: 1200px 이상 (풀 레이아웃)
- **Tablet**: 768px ~ 1024px (2컬럼 그리드)
- **Mobile**: 768px 이하 (단일 컬럼, 햄버거 메뉴)

모든 섹션이 모바일에서도 완벽하게 동작하며, 터치 인터랙션을 지원합니다.

- **Font Awesome** v6.4.0 - 아이콘
- **Google Fonts** - Noto Sans KR & Playfair Display

### 기술
- **HTML5**: 시맨틱 마크업

- **JavaScript (ES6+)**: 모듈화된 인터랙션
- **Responsive Design**: 모바일 우선 접근

## 🎨 커스터마이징 가이드

### 컬러 변경
`css/style.css` 파일의 `:root` 섹션에서 CSS 변수 수정:

```css
:root {
    --primary-color: #d4558d;      /* 메인 컬러 */
    --secondary-color: #7c5ccc;    /* 서브 컬러 */
    /* ... 기타 변수들 */
}
```

### 콘텐츠 수정
`index.html` 파일에서 텍스트 및 이미지 수정:
- 회사명: "뷰티디테일" 검색 후 변경
- 가격: `.plan-price .amount` 섹션에서 수정
- 이미지: Unsplash URL을 원하는 이미지로 교체
- 연락처: `#contact` 섹션에서 인스타그램 링크 및 이메일 수정

### 섹션 추가/제거
HTML에서 `<section>` 태그 단위로 추가/제거 가능하며, 해당 CSS 클래스도 함께 수정

## 🔧 주요 기능 설명

### 1. 플랜 선택 플로우
```javascript
플랜 카드 클릭 → 문의 폼으로 자동 스크롤 → 선택한 플랜 자동 입력 → 알림 표시
```

### 2. 카운터 애니메이션
히어로 섹션의 통계 숫자가 화면에 보일 때 0부터 타겟 숫자까지 카운트업

### 3. FAQ 아코디언
한 번에 하나의 질문만 열리며, 부드러운 애니메이션 제공

### 4. 알림 시스템
사용자 액션에 따라 화면 우측 상단에 알림 메시지 표시 (성공/오류/정보)

### 5. 모바일 메뉴
768px 이하에서 햄버거 메뉴로 전환, 클릭 시 전체 화면 메뉴 표시
